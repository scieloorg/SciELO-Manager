Notices API
===============

List all notices
----------------

Request:

  **API version 1**

  *GET /api/v1/collections*

  **API version 2**

  *GET /api/v2/collections*

Parameters:

  **--**

Optional Parameters:

  **callback**

    *String* of the callback identifier to be returned when using JSONP.

  **format**

    *String* of the desired output format. The options are **xml, json,
    yml**.


Allowed Method:

  GET, PUT, POST
